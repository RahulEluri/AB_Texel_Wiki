# Requesting Changes

## Overview

This page explains how to request new reports, modifications to existing reports, or other changes to the AB Texel reporting system.

## Change Request Types

### New Report Request
- Create a new report for specific business needs
- Estimated turnaround: 2-4 weeks
- Requires business case and requirements document

### Report Modification
- Change to an existing report (layout, metrics, data)
- Estimated turnaround: 1-2 weeks
- Submit change description and current report name

### Data Integration Request
- Add new data source to system
- Estimated turnaround: 4-8 weeks
- Requires IT assessment and security review

### Access Request
- Request new permissions or role change
- Estimated turnaround: 1-2 business days
- Manager approval required

## How to Submit a Request

### Step 1: Gather Requirements
Before submitting, prepare:
- Business justification
- Required metrics and dimensions
- Target audience/users
- Preferred format (Power BI, Excel, etc.)
- Timeline needs

### Step 2: Submit Request
Submit through one of these channels:
1. **Online Portal:** Access the change request system at [URL]
2. **Email:** Send to reporting-team@abtexel.com
3. **In Person:** Visit the Data Center (Building A, 3rd Floor)

### Step 3: Review & Assessment
- Initial review: 24-48 hours
- Feasibility assessment: 1-2 weeks
- Cost/resource estimation provided

### Step 4: Approval
- Business owner approval (if applicable)
- Budget verification
- Timeline confirmation

### Step 5: Development & Delivery
- Development begins after approval
- Testing period: 1-2 weeks
- User training provided if needed
- Go-live support included

## Request Prioritization

Requests are prioritized based on:
- **Urgency:** Business criticality
- **Complexity:** Technical difficulty
- **Resources:** Current team capacity
- **Alignment:** Strategic importance

## Change Request Form

**Form Fields:**
- Request Title
- Requestor Name & Department
- Business Justification
- Requirements Description
- Target Users
- Preferred Format
- Timeline Needed
- Budget Approval (if applicable)
