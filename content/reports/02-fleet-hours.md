# Fleet Hours Report

## Overview

The Fleet Hours report tracks vehicle utilization, driver hours, maintenance schedules, and overall fleet efficiency. Data comes from the BigMile fleet management system.

## Key Metrics

| Metric | Description | Update Frequency |
|--------|-------------|------------------|
| Total Fleet Hours | Combined hours of all vehicles | Hourly |
| Utilization Rate | Percentage of time vehicles are in use | Hourly |
| Driver Hours | Total hours driven by all drivers | Hourly |
| Idle Time | Time vehicles are not in use | Hourly |
| Maintenance Days | Scheduled maintenance downtime | Daily |
| Fuel Efficiency | Miles per gallon by vehicle | Hourly |

## Report Sections

### Fleet Overview
- Total vehicles and utilization rates
- Driver hours summary
- Average idle time
- Fuel efficiency metrics

### Vehicle Analysis
- Individual vehicle performance
- Utilization trends by vehicle
- Maintenance schedules
- Fuel consumption by vehicle

### Driver Analysis
- Driver hours and compliance
- Safety metrics
- Peak usage times
- Performance comparisons

### Efficiency Trends
- Monthly utilization trends
- Seasonal patterns
- Efficiency improvements
- Maintenance impact analysis

## How to Use

1. **Access:** Log in to Power BI and navigate to Fleet Hours
2. **Filter:** Select date range, vehicles, or drivers
3. **Analyze:** Review trends and drill into specific vehicles
4. **Export:** Download data for further analysis

## Key Features

- Real-time BigMile data integration
- Hourly data refresh
- Vehicle-level and driver-level drill-down
- Trend analysis and forecasting
- Email delivery available
