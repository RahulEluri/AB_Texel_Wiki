# How to Use Power BI Reports

## Getting Started

### Step 1: Access Power BI
1. Open a web browser (Chrome, Firefox, Safari, or Edge)
2. Go to [powerbi.abtexel.com](http://powerbi.abtexel.com)
3. Sign in with your AB Texel credentials
4. Navigate to the Apps section

### Step 2: Select a Report
- Click on the report you want to view
- Report will load in the browser
- Wait for data to fully load (typically under 5 seconds)

### Step 3: Explore the Dashboard

## Navigating Reports

### Dashboard Pages
- Most reports have multiple pages (tabs at the bottom)
- Click tabs to switch between different views
- Use breadcrumb navigation at the top to go back

### Filters & Slicers
- Filters appear on the right side of the report
- Click dropdown menus to select values
- Use date pickers for date ranges
- Apply multiple filters simultaneously

### Example Filter Workflow
1. Select Department: "Operations"
2. Select Date Range: "Last 30 Days"
3. Report automatically updates with filtered data

## Interacting with Visuals

### Click & Drill
- Click on data points (bars, dots, categories) to drill down
- Breadcrumb shows your navigation path
- Click the back button to go to previous level

### Hover for Details
- Hover over charts to see detailed values
- Tooltips show additional information
- Data updates as you move your cursor

### Sorting
- Click column headers to sort data
- Click again to reverse sort order
- Use sort indicators (up/down arrows)

## Common Tasks

### Export Data
1. Click the "..." (more options) menu
2. Select "Export data" or "Export this visual"
3. Choose format: Excel, PowerPoint, or PDF
4. Follow download instructions

### Print Reports
1. Click "File" → "Print"
2. Or use browser print (Ctrl+P)
3. Select printer and settings
4. Click Print

### Share Reports
1. Click Share button (top right)
2. Enter email addresses
3. Add optional message
4. Click Share

### Schedule Delivery
1. Click "..." (more options) menu
2. Select "Schedule a subscription"
3. Choose frequency: Daily, Weekly, Monthly
4. Select recipients
5. Confirm and subscribe

## Tips & Tricks

### Performance
- Use filters to reduce data volume
- Clear browser cache if slow
- Avoid opening too many tabs
- Use supported browsers for best performance

### Finding Data
- Use Ctrl+F (Find) to search within page
- Read axis labels and titles carefully
- Check filters to understand what data is shown
- Use drill-down for detailed analysis

### Understanding Visuals
- **Bar Charts** - Compare values across categories
- **Line Charts** - Show trends over time
- **Tables** - Display detailed data
- **Cards** - Show single metrics
- **Maps** - Show geographic data

## Troubleshooting

### Report Won't Load
1. Refresh your browser (F5 or Ctrl+R)
2. Clear browser cache and cookies
3. Try a different browser
4. Contact [support](../10-contact-support.md) if issue persists

### Data Looks Wrong
1. Check all applied filters
2. Verify date range is correct
3. See [Data Sources & Refresh Schedule](../02-data-sources.md) for timing
4. Report the issue to support@abtexel.com

### Permission Denied
- You may not have access to this report
- Contact support to request access
- See [Access & Permissions](../03-access-permissions.md)

## Mobile Access

Power BI reports are accessible on mobile devices:
1. Download Power BI mobile app (iOS or Android)
2. Sign in with your credentials
3. Find reports in the app
4. Note: Some features may be limited on mobile
